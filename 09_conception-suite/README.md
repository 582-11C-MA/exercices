# Exercice : Conception (suite)

Définissez chacune des fonctions suivantes. Pour chacune d'elle,
assurez-vous de suivre les étapes de conception.

-   Concevez une fonction nommée `voyelle?` qui prend comme argument une
    lettre et retourne `#true` seulement si la lettre donnée est une
    voyelle.

-   Concevez une fonction `bissextile?` qui prend comme argument une
    année et retourne `#true` si l'année en question est bissextile. Une
    année est bissextile seulement si elle respecte l'un des deux
    critères suivants : l'année est divisible par 4 sans être divisible
    par 100 ; et l'année est divisible par 400. Autrement dit : pour
    être bissextile, une année doit dans tous les cas être divisible par
    4 ; mais si c'est une année de centenaire (comme 1800, 1900, etc.),
    elle doit également être divisible par 400.

-   Concevez une fonction nommée `type-triangle` qui prend comme
    arguments la longueur des trois côtés d'un triangle, et retourne
    soit `"équilatéral"`, `"isocèle"` ou `"scalène"`. Un triangle
    équilatéral est un triangle dont tous les côtés sont égaux. Un
    triangle isocèle est un triangle dont deux côtés sont égaux. Un
    triangle scalène est un triangle dont les côtés sont inégaux.

-   Concevez une fonction nommée `consonne?` qui prend comme argument
    une lettre et retourne `#true` seulement si la lettre donnée est une
    consonne.

-   Concevez une fonction nommée `vingt-et-un` qui prend comme arguments
    deux mains, et retourne la main gagnante. Le principe du jeu
    vingt-et-un est que le total de ses cartes soit le plus près de 21,
    sans toutefois dépasser. Une main dont le total dépasse 21 est
    automatiquement perdante.
