# Exercice : La récursivité

Concevez les fonctions suivantes. Utilisez au moins une fois un
procéssus itératif. 

-   Une fonction qui, étant donnée une liste de saveurs de crème glacée,
    produit un élément HTML `<ul>` contenant toutes les saveurs.

-   Une fonction récursive qui détermine si une lettre donnée est une
    voyelle.

-   Une fonction qui retourne le dernier élément d'une liste non-vide
    donnée.

-   Une fonction qui construit une liste contenant une suite
    décroissante de nombres dont le premier nombre est donné, et le
    dernier nombre est 1. Par exemple, étant donné `3`, la valeur de
    retour de la fonction doit être `(list 3 2 1)`.

-   Une fonction qui construit une liste contenant une suite croissante
    de nombres dont le dernier nombre est donné, et le premier nombre
    est 1. Par exemple, étant donné `3`, la valeur de retour de la
    fonction doit être `(list 1 2 3)`.

-   Une fonction qui, étant donnée une liste d'étapes pour une recette
    culinaire, retourne une chaîne contenant les étapes numérotées. Par
    exemple, étant donnée la liste suivante `(list "Couper l'ail"
    "Ajouter l'huile" "Mélanger le tout")`, la valeur de retour doit
    être `"1. Couper l'ail 2. Ajouter l'huile 3. Mélanger le tout"`.
